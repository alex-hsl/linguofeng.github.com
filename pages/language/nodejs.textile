        <h3>一、安装Nodejs</h3>
    </div>

    <h4><small>1.1</small> ubuntu 12.10 x32</h4>
<pre>
$ sudo add-apt-repository ppa:chris-lea/node.js
$ sudo apt-get update
$ sudo apt-get install nodejs npm   # npm为Nodejs的包管理器
</pre>

    <h4><small>1.2</small> OS X</h4>
<pre>
$ brew install node
    add /usr/local/share/npm/bin to $PATH
</pre>

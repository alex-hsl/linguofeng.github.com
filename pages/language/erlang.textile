        <h3>二、Hello World</h3>
    </div>
    <p>@vim hello.erl@</p>
<pre class="prettyprint">
-module(hello).
-export([sayHello/0]).

sayHello() -> io:fwrite("hello, world\n").
</pre>

<pre class="prettyprint">
$ erl
1> c(hello).         // 加载hello.erl文件
{ok,hello}
2> hello:sayHello(). // 调用sayHello方法
hello, world
ok                   // 连续按两次ctrl + c退出
</pre>
